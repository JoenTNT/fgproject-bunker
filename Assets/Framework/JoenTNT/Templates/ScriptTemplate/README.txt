Script Template ini dibuat untuk mempercepat pekerjaan membuat skrip dengan menggunakan template Lentera Nusantara Framework.
Untuk menggunakan Template ini, lakukan langkah-langkah berikut:

1. Copy semua file .cs.txt yang ada di folder ini;
2. Buka folder utama dimana Editor Unity Game Engine anda ditempatkan;

Contoh: .../Unity/Hub/Editor/<versiUnity>/Editor/Data/Resources/ScriptTemplates

3. Setelah itu Paste ke folder tersebut;

Jika anda sedang membuka Unity sebelumnya, restart Unity Engine anda agar ScriptTemplate dapat di load.
